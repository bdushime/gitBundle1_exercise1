Git Practice Page

This is a simple HTML page for practicing Git commands and workflow. Here are some exercises you can try:

  • Create a new branch and modify this text.
  • Add a new feature like an image or a table.
  • Fix a typo and commit the change.
  • Merge changes from another branch.
  • Resolve a merge conflict.

Have fun exploring Git!
